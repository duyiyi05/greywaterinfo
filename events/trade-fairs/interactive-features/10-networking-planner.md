## Follow-up Takvimi

### Fuardan Döndükten Sonra

```
Gün 1-2 (fuar bitince):
  → Tüm kartviziteri fotoğrafla
  → Excel listesine gir (Firma / Kontak / Konu / Sıcaklık)
  → LinkedIn bağlantı isteği gönder (hepsi)

Gün 3-5:
  → 🔴 Sıcak kontaklar: Kişisel email gönder
    Konu: "Great meeting you at [FUAR] — next steps"
    İçerik: Kısa özet + fiyat teklifi / numune bildirimi

  → 🟡 Ilık kontaklar: Genel email gönder
    Konu: "Following up from [FUAR]"
    İçerik: Şirket broşürü + web sitesi

  → 🔵 Soğuk kontaklar: LinkedIn mesaj yeter

Hafta 2:
  → Yanıt verenlerle video görüşme planla
  → Numune gönderilecek Canton tedarikçilerine sipariş ver

Ay 1:
  → Teklif gönder (aktif leadler)
  → Distribütör sözleşme taslağı gönder (ilgilenenler)

Ay 2-3:
  → Pilot sipariş teklifleri
  → Yerel ziyaret / Türkiye'ye davet
```

## CRM Entegrasyonu

### Basit CRM (Excel/Google Sheets)

**Sütunlar:**
```
A: #
B: Ad Soyad
C: Firma
D: Ülke
E: Sektör (Tedarikçi / Distribütör / Müşteri / Partner)
F: Pozisyon
G: Email
H: Tel/WhatsApp
I: LinkedIn URL
J: WeChat (Canton tedarikçileri için)
K: Fuar
L: İlk Temas Tarihi
M: Konu / Ürün
N: Sıcaklık (Sıcak/Ilık/Soğuk)
O: Son Görüşme Tarihi
P: Durum (Yeni / Email Gönderildi / Görüşme / Teklif / Kapandı / Hayır)
Q: Tahmini Değer (€)
R: Not
```

### Durum Akışı
```
Yeni → Email Gönderildi → Görüşme → Teklif → Kapandı (✅ veya ❌)
```
