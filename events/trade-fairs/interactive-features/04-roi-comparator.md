# ÖZELLİK 4 — ROI Karşılaştırıcı
**Her fuar için 3 senaryo: İyimser / Gerçekçi / Kötümser**

---

## ROI Hesaplama Formülü

```
ROI (%) = ((Gelir - Maliyet) / Maliyet) × 100

Gelir kaynakları:
  - Direkt satış (fuar sırasında)
  - 12 ay içinde kapanan leadler
  - Tedarikçi değişimi tasarrufu
  - Teknoloji/bilgi kazanımı (dolaylı)
```

## Kümülatif 2 Yıl ROI Analizi

```
YATIRIM (2026-2027):
  Aquatech 2026 stantsız  €3,100
  Canton 2026 (Kişi-1)   €1,750
  IFAT 2026 stantlı      €9,240
  Aquatech 2027 stantlı  €7,400
  IFAT 2027 stantlı      €10,790
  ─────────────────────────────
  TOPLAM                  €32,280

GELİR (Gerçekçi, 2026-2027):
  Canton tasarrufu       €16,000 (2 yıl)
  IFAT 2026 satışı      €144,000
  IFAT 2027 satışı      €200,000
  Aquatech 2027 satışı  €80,000
  ─────────────────────────────
  TOPLAM                €440,000

ROI 2 YIL = ((440,000 - 32,280) / 32,280) × 100 = +1,262%
```
